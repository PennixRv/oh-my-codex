# Git Master Command

Routes to the git-master agent for git operations.

## Usage

```
/git-master <git task>
```

## Routing

```text
Route the task to the `git-master` role or native agent surface.
```

## Capabilities
- Atomic commits with conventional format
- Interactive rebasing
- Branch management
- History cleanup
- Style detection from repo history
